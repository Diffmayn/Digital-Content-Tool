‚ú® Key Features

  • Role-Based Access Control: Each user sees only what they're allowed to access
  • Real-Time Metrics: Dashboard with live order statistics
  • Advanced Filtering: Filter by status, photographer, purchase group, priority, date range
  • Sample Management: Barcode scanning system for tracking physical articles
  • Image Upload: Photographers can upload deliverables directly
  • Brief Management: Agency users can attach detailed creative briefs
  • Export Functionality: Download filtered data as CSV
  • Responsive Design: Works on desktop, tablet, and mobile
  • Keyboard Shortcuts: Ctrl+K (search), Ctrl+M (scan), Esc (close)
